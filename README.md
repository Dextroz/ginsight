# ginsight
A tiny wrapper for Google's PageSpeed Insight API.

## Dependencies
```
pip3 install -r requirements.txt
```

## Example Usage
```
    import g_insight
    insight = g_insight("Insert API Key Here.")
    print(insight.check_website("https://google.com"))
```
